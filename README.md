# Словник — ქართული

Локальний Flask-сервер для запису грузинських слів з телефону, поки вчу мову на LingWing.

## Запуск

```
pip install -r requirements.txt
python app.py
```

Або подвійний клік на `start.bat`.

Консоль покаже дві адреси:
- `http://127.0.0.1:5000` — з цього комп'ютера
- `http://<local-ip>:5000` — з телефону (той самий Wi-Fi). IP може змінюватись після перезавантаження роутера — дивись актуальний в консолі при кожному старті.

З телефону відкрий адресу в браузері і додай на головний екран ("Add to Home Screen") — буде як окрема аплікація.

При першому запуску Windows Firewall може спитати дозвіл для Python — треба дозволити для приватних мереж, інакше телефон не з'єднається.

## Що вміє

- Додавання слова: грузинська, переклад, приклад/нотатка (необов'язково)
- Пошук по грузинській і перекладу
- Редагування та видалення
- Експорт у CSV (кнопка "⬇ CSV") — щоб потім віддати AI на перевірку правильності перекладів/прикладів
- Немає кешування сторінок (`Cache-Control: no-store`) — на телефоні завжди актуальна версія без потреби перевстановлювати шорткат

## Структура

- `app.py` — сервер і вся логіка (SQLite, роути)
- `templates/index.html` — головна сторінка (форма додавання, пошук, список)
- `templates/edit.html` — сторінка редагування слова
- `dictionary.db` — SQLite база, створюється автоматично при першому запуску, **не в git**
- `start.bat` — запуск подвійним кліком на Windows

## База даних

Таблиця `words`: `id, georgian, translation, example, created_at`. Схема створюється автоматично (`init_db()` в `app.py`), міграцій поки немає.
